# Airesto Reservations

Страница просмотра бронирований ресторана на Vue 3.
Ссылка: https://airestoru.netlify.app/

## Возможности

- Получение данных через моковый API-слой.
- Переключение доступных дат.
- Фильтрация отображаемых зон.
- Sticky-заголовки столов и времени.
- Время ресторана с учетом timezone.
- Отображение заказов, банкетов, бронирований и живой очереди.
- Hover-состояния карточек событий.
- Темная/светлая тема.
- Режим скрытия персональных данных.
- Выделение времени и столов с выводом результата в консоль по кнопке `Создать`.

## Запуск

Вариант без установки зависимостей, удобный для быстрой проверки:

```bash
node scripts/static-server.js
```

После установки зависимостей можно использовать Vite:

```bash
npm install
npm run dev
```
